자주 사용하는 타임리프(Thymeleaf)의 3가지 속성

1. 분기문 속성

  • if문, else if문 등
    th:if="${question != null}"

2. 반복문 속성

  • for each문과 유사
    th:each="question : ${questionList}"
  • loop 객체 활용
    th:each="question, loop : ${questionList}"
    • loop.index: 루프 반복 순서, 0부터 1씩 증가
    • loop.count: 루프 반복 순서, 1부터 1씩 증가
    • loop.size: 반복 객체의 요소 개수(ex. questionList의 요소 개수)
    • loop.first: 루프의 첫 번째 요소면 true
    • loop.last: 루프의 마지막 요소면 true
    • loop.odd: 루프의 홀수 번째 요소면 true
    • loop.even: 루프의 짝수 번째 요소면 true
    • loop.current: 현재 대입된 객체와 동일(ex. question))

3. 텍스트 속성

  • 해당 요소의 텍스트값 출력
    th:text="${question.subject}"
  • 이렇게도 출력 가능
    [[${question.subject}]] [[${question.createDate}]]
